# Changelog

All notable changes to the Click E-Commerce System are documented in this file.
This project uses Git tags for each release so progress can be reviewed per milestone.

The format is loosely based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to semantic-style version tags.
